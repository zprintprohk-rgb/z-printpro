## 🧪 手动验证步骤

### 步骤 1：检查页面加载
1. 在浏览器中查看已打开的三个标签页
2. 确认每个页面都显示正常内容（非 404 或错误页面）

### 步骤 2：检查浏览器控制台
1. 按 `F12` 打开开发者工具
2. 切换到 **Console** 标签
3. 检查是否有红色错误
4. 特别关注：
   - Supabase 连接错误
   - API 请求失败
   - 资源加载失败

### 步骤 3：测试 Supabase 连接
1. 访问产品页面
2. 检查产品数据是否从 Supabase 加载
3. 如看到产品列表，说明连接正常

### 步骤 4：测试语言切换
1. 在任意页面点击语言切换器
2. 确认能切换到其他语言版本
